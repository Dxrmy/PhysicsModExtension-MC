**PhysicsModExtension** expands upon traditional Minecraft physics to introduce realistic structural integrity and object momentum. Walls collapse, blocks fall with force, and the world feels weightier.

## Features
- 🏗️ **Structural Integrity**: Buildings now require support; clearing pillars can lead to catastrophic collapses.
- ☄️ **Momentum Simulation**: Enhanced logic for falling blocks and entity collisions.
- 📉 **Optimized Core**: Custom math library to ensure physics calculations don't impactTPS.
- 🔧 **Deep Customization**: Adjust gravitational constants and integrity limits via config.

## Tech Stack
- **Mod Loader**: Fabric API (1.21.11)
- **Math**: Custom Vector-Calculation Engine
- **Hooks**: Mixin-based Physics Overrides
