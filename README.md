**PhysicsModExtension** is a specialized addon for the popular Physics Mod, focusing exclusively on expanding and refining the **Structural Integrity** system.

## Enhancements
- 🏗️ **Deep Integrity Logic**: Advanced calculations for how blocks support one another.
- 📉 **Stress Distribution**: Realistic collapse patterns based on load-bearing pillars and structural weak points.
- ⚖️ **Integration**: Designed to work perfectly alongside the Physics Mod's core systems.
